## ğŸ“ FREQUENTLY ASKED QUESTIONS

**Q: How long until a child levels up?**
A: Typically 5-8 projects to reach Level 2, then more as you go higher. This encourages sustained engagement.

**Q: Can children see other children's progress?**
A: No. Each child has their own private growth map. The system is non-competitive.

**Q: What if a project has no skill mapping?**
A: Default values are used (30 making, 20 thinking, 20 problem-solving, etc.). You can customize anytime.

**Q: Will this work on mobile?**
A: Yes! All templates are fully responsive. Tested and optimized for mobile devices.

**Q: Can parents see progression?**
A: Currently parents see their child's profile. A full parent dashboard is planned for v1.1.

**Q: How do I change the reflection bonus?**
A: Edit the `add_points()` method in `GrowthPathway` model. Change `points * 0.25` to your desired percentage.
